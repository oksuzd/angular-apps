# Angular apps

This app presents some training projects that were developed on Angular v.15, v.16.

1. A table with users that can only be accessed by the logged in user. I used API from reqres.in and AG Grid instrumentary.
2. Basic expansion panel with simple animation and API from reqres.in.
3. The weather app with various API: api.ipgeolocation.io, api.geoapify.com, api.weatherapi.com, pixabay.com.
